This example demonstrates how to drag and drom items onto the timeline from the outside (an external list, etc). Add p:ajax with event="drop" to the timeline tag to activate the drag-and-drop feature. The AJAX listener gets an instance of the class TimelineDragDropEvent. Besides start / end date and group, this event object also contains a client ID of the dragged component and dragged model object if draggable item is within a data iteration component.

Drag events from the list on the left side and drop them onto the timeline!
