This example demonstrates configurable options such as "axisOnTop", "editable", "selectable", "zoomable", "moveable", "eventStyle", "showCurrentTime", "showButtonNew", "showNavigation" and some other.

Note: If the attribute "editable" is true, the events can be created, edited (content), changed (start / end time) and deleted. Events can only be editable when the option selectable is true (default). When editable is true, the timeline can fire events "change", "edit", "add", "delete". This global setting "editable" can be overwritten for individual events by setting a value in field "editable".

How to create, edit, change and delete events? When the timeline is defined as editable, events can be moved to another time by dragging them. By double clicking, the contents of an event can be edited. An event can be deleted by clicking the delete button on the upper right. A new event can be added in different ways: by double clicking in the timeline, or by keeping the Ctrl key down and clicking or dragging in the timeline, or by clicking the add button in the upper left of the timeline (if "showButtonNew" is set to true), and then clicking or dragging at the right location in the timeline.
