When the property group in TimelineEvent is provided, all events with the same group are placed on one line. A vertical axis is displayed showing the groups. Grouping events can be useful for example when showing availability of multiple people, rooms, or other resources next to each other. Note, that groupped events will not be stacked automatically, when they overlap each other.

Besides grouping, this example also demonstrates how to merge events when they get overlapped by dragging and dropping them. Please consider the attribute groupsChangeable.
