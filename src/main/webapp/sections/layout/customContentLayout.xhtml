To place any custom content between layout panes there are two alternative ways:

1) Place a nested layout panes (pe:layoutPane) within a div element with the class "ui-layout-content". This way is applied in this example.
2) Add the option "containerSelector" to child options (see LayoutOptions class). The value of "containerSelector" points to a wrapper-div element which acts as a container for nested layout panes.
