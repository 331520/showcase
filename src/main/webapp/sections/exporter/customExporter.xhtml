This feature enables fully customizable own exporter with the required functionalities using Service Loader Concept.

Steps required:

  • Step 1: Create a folder named META-INF under resouces folder.Below META-INF folder create another folder called services.
  • Step 2: Creae a file name with ExporterFactory as a service(Fully binary name of the service).
        Ex: org.primefaces.extensions.component.exporter.ExporterFactory.
  • Step 3: Provide your own implementaions/providers of Exporter factory in your project.
        Ex: org.primefaces.extensions.showcase.util.CustomExporterFactory.
  • Step 4: Copy the exporter implementations and add your own changes.
