The requiredLabel extends the h:ouputLabel and adds an required indicator.

The required indicator can be changed with the requiredIndicator attribute and can be styled via css: .pe-required-label-indicator.

Also if the "for" component is invalid, an error style class will also be rendered: .re-required-label-invalid. <p:outputPanel id="container"> <pe:requiredLabel for="loginName" value="LoginName:" /> <p:inputText id="loginName" /> <p:commandButton id="submitLoginName" value="Post" process="container" update="container" /> </p:outputPanel>
