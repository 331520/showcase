PrimeFaces Extensions provides a separate tag for pagination in order to reuse (share) one and the same paginator settings between multiply components requiring pagination. Place pe:paginator within ui:include to reuse it.

pe:paginator can be applied to all PrimeFaces components extended from UIData (p:dataTable, p:dataGrid, p:dataList).
